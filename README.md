# Analise-Financeira
O este projeto foi desenvolvido com o objetivo de simular uma situação real de análise de dados no contexto de uma rede de lojas. A proposta consistiu em analisar indicadores de faturamento, desempenho de vendas e eficiência operacional para identificar a unidade com menor performance.

A partir dos dados processados e dos gráficos gerados, foi possível comparar as lojas com base em métricas como faturamento total, volume de vendas por categoria e avaliação dos clientes. Com isso, tornou-se viável identificar a loja com menor eficiência e, com base em evidências quantitativas, apresentar uma recomendação final fundamentada nos resultados obtidos.

O projeto reforça a importância da análise de dados como ferramenta estratégica para apoiar a tomada de decisão e otimizar os resultados do negócio.
